The problem

Why agents break the old security model

01 · Access
Broad, indiscriminate access
  • Reasons over every available tool, chains them unpredictably
  • Reaches everything — not the minimum needed
02 · Trust
Blind instruction-following
  • A poisoned doc or MCP server becomes an instruction
  • Black-box reasoning — hard to audit, harder to reconstruct
03 · Blast radius
Scale amplification
  • One compromised agent touches hundreds of systems in an hour
  • No manual effort to spread
04 · Identity
Attribution gap
  • Who authorized the action vs who executed it?
  • IAM wasn't built to answer that in real time

The problem

So: red-team it

Red-teaming — deliberately attacking your own agent to find the weaknesses before an adversary does. The moves a manual spot-check never makes:

Multi-turn
Escalation
  • Each message looks benign
  • Attack assembles across turns — invisible to single-turn checks
Indirect
Injection
  • Attacker controls what the agent reads — emails, docs, search
  • You never see it
Persistent
Memory poisoning
  • Payload survives session resets
  • Future users inherit the attack
Many-shot
Jailbreaking
  • 100+ fabricated in-context examples
  • Gradually normalize the behavior you forbade

Resolutions

Your agent misbehaves — now what?

OUTER = BROADEST CONTROL  ·  INNER = MODEL-LEVEL

System-wide authn/authz — identity + least privilege; a hijacked agent still can't reach what it shouldn't
Deterministic tool guardrails — allow-lists, argument validation, confirmation gates, scopes. The non-LLM safety net
Agent guardrails — input/output guardrails + judges-in-the-loop (clarabelle-still-a-cow as a runtime guard)
Stronger model — more-aligned models resist more (cost ↑, latency ↑; no guarantee)
Swap model provider — same prompt, different provider; robustness to an attack class varies
Your agent
Defense-in-depth. No single silver bullet.

Get started

Wire it into your agent

from evaluatorq.redteam import red_team

report = await red_team(
    target=MyAgent(),                 # any send_prompt + reset_conversation
    vulnerabilities=["goal_hijacking", "tool_misuse"],
    mode="dynamic", max_turns=4,
)
print(f"Resistance: {report.summary.resistance_rate:.0%}")
